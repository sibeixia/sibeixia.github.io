# 珠宝数字产品护照 - 使用说明

## 文件结构

为了确保页面可以正常打开，请保持以下文件结构：

```
项目文件夹/
├── index-standalone.html  (主HTML文件)
└── images/                (图片和视频文件夹)
    ├── jewelry-front.png  (正面图片)
    ├── jewelry-back.png   (背面图片)
    ├── jewelry-side.png   (侧面图片)
    └── jewelry-3d.mp4     (3D动画视频)
```

## 使用方法

1. **确保文件结构正确**
   - `index-standalone.html` 和 `images` 文件夹必须在同一目录下

2. **打开页面**
   - 双击 `index-standalone.html` 文件即可在浏览器中打开
   - 或者右键选择"用浏览器打开"

3. **移动文件夹**
   - 可以将整个项目文件夹移动到任何位置
   - 只要保持 `index-standalone.html` 和 `images` 文件夹的相对位置不变即可

## 注意事项

- 所有路径都是相对路径，不依赖绝对路径
- 不要单独移动 `index-standalone.html` 文件，必须和 `images` 文件夹一起移动
- 如果图片或视频不显示，请检查文件是否在 `images` 文件夹中，文件名是否正确

## 更新内容

如果需要更新图片或视频：
1. 将新文件放入 `images` 文件夹
2. 确保文件名与代码中的文件名一致
3. 刷新浏览器页面即可看到更新
