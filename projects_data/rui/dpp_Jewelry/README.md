# Jewelry Digital Product Passport

A modern web application for luxury jewelry digital product passports, featuring product information, authentication, supply chain tracking, sustainability information, and more.

## Features

- **Home Page**: Product overview with navigation to different sections
- **Digital Twin**: 3D product visualization and NFT trading information
- **Authentication**: Product verification and ownership tracking
- **Product Info**: Detailed product specifications and features
- **Supply Chain**: Material sourcing and carbon emission tracking
- **Sustainability**: Environmental certifications and commitments
- **Care Guide**: Care instructions and recycling program

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

## Technology Stack

- React 18
- React Router DOM
- Vite
- CSS3
