# 图片和视频文件

请将所有图片和视频文件放在此 `images` 文件夹中。

## 需要的文件

### 图片文件
- `jewelry-front.png` - 珠宝正面图片
- `jewelry-back.png` - 珠宝背面图片  
- `jewelry-side.png` - 珠宝侧面图片

### 视频文件
- `jewelry-3d.mp4` - 3D动画视频（必需）
- `jewelry-3d.webm` - 3D动画视频（可选）
- `jewelry-3d.mov` - 3D动画视频（可选）

## 文件结构

确保文件结构如下：
```
项目文件夹/
├── index-standalone.html
└── images/
    ├── jewelry-front.png
    ├── jewelry-back.png
    ├── jewelry-side.png
    └── jewelry-3d.mp4
```

**重要提示：** 
- `index-standalone.html` 和 `images` 文件夹必须在同一目录下
- 这样无论将整个文件夹移动到任何位置，页面都能正常打开
