# 图片和视频说明

请将以下文件放置在此目录（`public/images/`）中：

## 需要的图片文件

1. **jewelry-front.png** - 珠宝正面图片（用于首页和Product Info页面主图）
2. **jewelry-back.png** - 珠宝背面图片（用于Product Info页面缩略图）
3. **jewelry-side.png** - 珠宝侧面图片（用于Product Info页面缩略图）

## 需要的3D动画视频文件

1. **jewelry-3d.mp4** - 3D动画视频（MP4格式，推荐）
2. **jewelry-3d.webm** - 3D动画视频（WebM格式，可选）
3. **jewelry-3d.mov** - 3D动画视频（MOV格式，可选）

## 文件要求

### 图片要求
- 格式：PNG 或 JPG
- 建议尺寸：至少 800x800 像素（正方形）
- 背景：建议使用白色或透明背景
- 文件大小：建议每个文件不超过 2MB

### 视频要求
- 格式：MP4（推荐）、WebM 或 MOV
- 建议分辨率：1920x1080 或更高
- 建议帧率：30fps 或 60fps
- 文件大小：建议不超过 50MB（可根据需要调整）
- 编码：H.264（MP4）或 VP9（WebM）

## 使用方法

1. 将图片和视频文件重命名为上述文件名
2. 将文件放置在 `public/images/` 目录中
3. 刷新浏览器即可看到内容

如果文件不存在，页面会自动显示占位符。
