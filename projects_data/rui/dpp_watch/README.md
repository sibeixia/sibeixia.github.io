# 数字产品护照 - 奢侈手表

这是一个数字产品护照（Digital Product Passport, DPP）网页应用，用于展示奢侈手表的详细信息。

## 功能特性

- **首页**：展示产品图片和主要信息按钮
- **数字孪生**：3D产品展示、社交媒体分享、NFT交易信息
- **产品认证**：产品验证状态、品牌声明、所有权信息
- **产品信息**：详细的产品规格、颜色选择、尺寸选择
- **供应链**：供应链信息和碳排放数据
- **可持续性**：环保信息、认证和承诺
- **保养指南**：保养说明和回收计划

## 文件结构

```
dpp watch/
├── index.html              # 首页
├── digital-twin.html       # 数字孪生页面
├── authentication.html     # 产品认证页面
├── product-info.html       # 产品信息页面
├── supply-chain.html       # 供应链页面
├── sustainability.html     # 可持续性页面
├── care-guide.html         # 保养指南页面
├── styles.css              # 样式文件
└── README.md               # 说明文档
```

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击各个信息按钮进入对应的详细页面
3. 每个页面都有"返回"按钮可以回到首页

## 技术说明

- 纯HTML、CSS实现，无需服务器
- 响应式设计，支持移动端和桌面端
- 所有页面共享统一的样式文件
- 简洁现代的UI设计

## 浏览器支持

支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）
