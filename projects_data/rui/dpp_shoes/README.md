# 鞋子数字产品护照 (Digital Product Passport - DPP)

一个现代化的数字产品护照网页应用，用于展示鞋类产品的完整生命周期信息，包括材料来源、制造过程、环境影响和生命周期追踪。

## 功能特性

- 📦 **产品信息展示** - 完整的产品基本信息、ID和标签
- 🧵 **材料追踪** - 详细的材料组成、来源、认证信息
- 🏭 **制造过程** - 生产工厂、认证标准、生产工艺
- 🌍 **环境影响** - 碳足迹、用水量、废料减少、可再生能源使用
- 🔄 **生命周期追踪** - 从原材料采购到回收再利用的完整生命周期

## 技术栈

- **HTML5** - 语义化标记
- **CSS3** - 现代化样式设计，响应式布局
- **JavaScript (ES6+)** - 交互功能和数据展示
- **Google Fonts** - Inter字体

## ⚠️ 重要：文件可移动性

**所有文件已配置为可移动使用！**

✅ **关键特性：**
- 所有路径都是相对路径
- 可以移动到任何位置（桌面、U盘、其他电脑等）
- 只要保持文件夹结构不变，网站就能正常工作

✅ **移动文件夹时的注意事项：**
1. **必须整个文件夹一起移动** - 不要单独移动文件
2. **保持文件夹结构** - images 和 videos 文件夹必须与 HTML 文件在同一目录
3. **所有文件必须在同一文件夹中** - 不要分散文件

✅ **验证方法：**
移动文件夹后，双击 `index.html` 文件：
- 如果页面正常显示 → 配置正确 ✅
- 如果图片不显示 → 检查 images 文件夹是否存在
- 如果视频不显示 → 检查 videos 文件夹是否存在
- 如果样式丢失 → 检查 styles.css 文件是否存在

## 使用方法

### 方法1：直接打开（推荐）
1. 双击 `index.html` 文件
2. 网站会在默认浏览器中打开

### 方法2：使用本地服务器（可选）
如果需要通过服务器访问：
```bash
# 使用Python
python -m http.server 8000

# 使用Node.js (需要安装http-server)
npx http-server
```
然后在浏览器中访问 `http://localhost:8000`

## 页面结构

### 1. 产品概览
- 产品名称和ID
- 可持续性标签
- 关键指标（碳足迹、可回收率）

### 2. 材料信息标签页
展示产品的材料组成，包括：
- 材料名称和百分比
- 材料来源
- 认证信息
- 可回收性

### 3. 制造过程标签页
展示生产相关信息：
- 生产工厂信息
- 认证标准
- 生产工艺说明

### 4. 环境影响标签页
展示环境相关指标：
- 碳足迹
- 用水量
- 废料减少率
- 可再生能源使用率

### 5. 生命周期标签页
展示产品从原材料到回收的完整生命周期时间线

## 自定义数据

要修改产品数据，请编辑 `script.js` 文件中的 `productData` 对象。你可以修改：

- 产品名称和ID
- 材料列表和详细信息
- 制造信息
- 环境指标
- 生命周期阶段
