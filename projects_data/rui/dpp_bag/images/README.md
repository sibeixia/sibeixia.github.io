# 图片和视频文件夹

请将您的包包图片和视频放在这个文件夹中。

## 需要的文件：

### 主页：
1. **main-black.jpg** - 主页产品主图（黑色包包）

### Digital Twin页面：
2. **digital-twin.mp4** - Digital Twin页面视频文件

### Product Info页面（每个颜色需要正反两面）：
3. **black-front.jpg** - 黑色包包正面
4. **black-back.jpg** - 黑色包包背面
5. **brown-front.jpg** - 棕色包包正面
6. **brown-back.jpg** - 棕色包包背面

## 文件要求：

### 图片：
- 建议尺寸：800x800像素或更大
- 格式：JPG、PNG、WebP
- 文件大小：建议每个文件不超过2MB

### 视频：
- 格式：MP4、WebM
- 文件大小：建议不超过10MB

## 使用方法：

1. 将您的图片和视频文件重命名为上述文件名
2. 放入此 `images` 文件夹
3. 刷新浏览器即可看到图片和视频

## 自定义文件名：

如果您想使用不同的文件名，可以修改 `app.js` 文件中对应的路径。
